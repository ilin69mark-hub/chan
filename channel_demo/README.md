# Демонстрация работы с двумя каналами в Go

Эта программа демонстрирует работу с двумя каналами сообщений из целых чисел в Go с следующими характеристиками:

- Прием сообщений из каналов никогда не приводит к блокировке
- Вероятность приема сообщения из первого канала в 2 раза выше, чем из второго
- Реализован механизм для демонстрации этого соотношения

## Как запустить программу

1. Убедитесь, что у вас установлен Go (версия 1.16 или выше рекомендуется)
2. Перейдите в директорию проекта:
```bash
cd /workspace/channel_demo
```

3. Запустите программу:
```bash
go run main.go
```

## Как работает программа

Программа создает два буферизованных канала для передачи целых чисел. Затем запускаются две горутины, которые отправляют данные в каждый из каналов. Основная логика получения данных реализует соотношение 2:1 в пользу первого канала с помощью случайного выбора и оператора select с default-веткой для предотвращения блокировки.

Канал 1 заполняется быстрее (каждые 5 мс), чем канал 2 (каждые 10 мс), что увеличивает вероятность его доступности. При получении данных, когда оба канала недоступны, используется вероятностный выбор: с вероятностью 2/3 (rand.Intn(3) < 2) пробуется канал 1, и с вероятностью 1/3 - канал 2.

Результаты выводятся в реальном времени, а в конце показывается статистика по количеству полученных сообщений из каждого канала и фактическое соотношение.
